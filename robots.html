Each virtual robot runs the real Locate → Poll → Draw loop in your browser — checking in to /api/robots/checkin once a second, taking whatever job the coordinator stages, then simulating the draw + exit. No admin token needed (check-in is the robot protocol). A robot only gets work if its name matches a region's robot field, so assign regions in the canvas config below first.
